# Open Learning Portal

A simple multi-page learning platform with login, course videos, and quiz.

## Features
- User registration & login (IndexedDB)
- Python course with video lessons
- Quiz system with scoring
- Clean minimal UI

## Quick Start (Windows - Recommended)

Double-click:
start.bat

This will:
- Start a local server
- Open the portal in your browser automatically

---

## Manual Run (All Systems)

python -m http.server

Then open:
http://localhost:8000
